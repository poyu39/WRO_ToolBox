# WRO_ToolBox
 WRO競賽組常用程式&工具範例
 
  ![image](https://user-images.githubusercontent.com/42506064/188401832-3e9fdda2-bec1-4fc1-b769-0c234197662d.png)

        
內容
  
    Ramp_Up_Down (緩加減)
  
    Sync (同步控制)
  
    PDLineFollower (循跡-PD控制版本)

    ColorMapping (映射色感)

    SetLight (映射光感)

    Sin_Ramp (三角函數緩加減)

    Direction_Turn (方向座標)

程式內有使用到台灣OFDL所開發的外掛方塊

    EV3 ToolBOX 
  
    Advanced Encoder

    Advanced RawMotor

    EV3 Color Sensor Advanced
